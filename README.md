PasswordGenerator
=========

Generate random passwords

#### Config
```go
Lower Case Letters: abcdefghijklmnopqrstuvwxyz
Upper Case Letters: ABCDEFGHIJKLMNOPQRSTUVWXYZ
Numbers: 0123456789
Special Characters: !@#$%&*()-=+
```

#### Example
```go
fmt.Println(randomSequence(10))
```

##### Output
```
C1n0Klg*LE
```

##### Output 2
```
w%ES1l-2Ki
```
